Lucas Chess
===========

Lucas Chess is a GUI of chess:

1. To learn to play chess.
2. To play chess against engines.

Dependencies
------------

* Python 2.7
* PyQt4
* PyAudio
* psutil
* Python for windows extensions
* pygal
* chardet
* python-chess
* pyllow
* photohash
* cython
